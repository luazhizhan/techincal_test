# Backend

This is the backend of the project. It is Nodejs with Typescript, Express.js, and Multer for file uploads.

## Installation

```bash
npm install
```

## Running

```bash
npm run dev
```

## Testing

```bash
npm run test
```

OR

```bash
npm run test:watch
```
